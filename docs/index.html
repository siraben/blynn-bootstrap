Portable TinyCC Bootstrap

x86_64 JSLinux booting Alpine Linux to a shell. Run bootstrap to build from the hex0 seed to a bootstrapped tcc.

Reboot 768 MB Shell Config

Milestones

  1. Kernel starts with console=hvc0 root=/dev/vda rw init=/init.
  2. /init prepares Alpine and drops to a shell.
  3. Run bootstrap to rebuild stage0-posix tools from the hex0 seed, then run the blynn/HCC portable chain.
  4. After the build, the shell has the bootstrapped compiler as blynn-tcc.

Shell Check

printf '#include <stdio.h>
int main(void){ puts("tcc works"); return 0; }
' >/tmp/tcc-smoke.c
command -v blynn-tcc
blynn-tcc -run /tmp/tcc-smoke.c
