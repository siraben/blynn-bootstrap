Portable TinyCC Bootstrap

x86_64 JSLinux booting Alpine Linux and running the repo's portable blynn bootstrap from the hex0 seed to a tcc shell.

Reboot Quick Shell Config

Milestones

  1. Kernel starts with console=hvc0 root=/dev/vda rw init=/init.
  2. /init runs /bootstrap/run-portable-demo.sh by default.
  3. The demo rebuilds stage0-posix tools from the hex0 seed, then runs the blynn/HCC portable chain.
  4. After the build, the shell has the bootstrapped compiler as blynn-tcc.

Shell Check

printf '#include <stdio.h>
int main(void){ puts("tcc works"); return 0; }
' >/tmp/tcc-smoke.c
command -v blynn-tcc
blynn-tcc -run /tmp/tcc-smoke.c
