Portable TinyCC Bootstrap

x86_64 JSLinux booting Alpine Linux and running the repo's portable blynn bootstrap to a tcc shell.

Reboot Full Build Config

Milestones

  1. Kernel starts with console=hvc0 root=/dev/vda rw init=/init.
  2. Alpine installs the Nix-built HCC TinyCC as blynn-tcc.
  3. The full in-guest portable chain is available with blynn_full=1 or /bootstrap/run-portable-demo.sh.

Shell Check

printf '#include <stdio.h>
int main(void){ puts("tcc works"); return 0; }
' >/tmp/tcc-smoke.c
command -v blynn-tcc
blynn-tcc -run /tmp/tcc-smoke.c
