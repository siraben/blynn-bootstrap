Portable TinyCC Bootstrap

riscv64 JSLinux booting Alpine Linux to a shell. Use the 1536 MB bootstrap VM and run bootstrap to build from the hex0 seed to a bootstrapped tcc.

Reboot 256 MB Shell 1536 MB Bootstrap Config

Milestones

  1. Kernel starts with normal Linux boot messages on console=hvc0, then runs init=/init.
  2. /init prepares Alpine and drops to a shell.
  3. Run bootstrap in the 1536 MB VM to rebuild stage0-posix tools from the hex0 seed, then run the blynn/HCC portable chain.
  4. After the build, the shell has the bootstrapped compiler as blynn-tcc.
